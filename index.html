Supported Languages¶

Makeup currently supports the following languages (in alphabetical order).

  • C
  • Diff
  • EEx
  • Elixir
  • Erlang
  • HEEx
  • HTML
  • JSON

Documentation¶

Makeup is documented on hexdocs. This page is not meant to serve as documentation. It's merely a demo of the several lexers and styles.

The formatter is the HTMLFormatter.
